Star Catcher is a small arcade-style game I solely built with Node, JavaScript, and TypeScript. It makes use of HTML5's Canvas web API and dynamically scales to the width of the page without losing quality.

Touchscreen: Tap and hold to move.

Catch stars to get points, but avoid everything else! Hit Game Start to begin.

You can find the code in my portfolio GitHub repo under the "ts" folder.
